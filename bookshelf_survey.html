Saker BookShelf

We are asking our readers to submit the titles and authors of their favorite books. These will be reviewed and selected books will be included in the BookShelf.

Help Page

This tool allows a person to recommend a book to the Saker BookShelf. In addition it allows an individual to maintain their own library of books and sub divided by bookshelfs if desired. Google books api is used to finds and obtain book info. Google has quotas so at times this tools may fail to work if we exceed our daily quota.
The Saker database will contain all books recommended by our readers and will be used to help select the top 50 books.

Look up Book and Enter book in database

1. Enter either title,author or isnb number

2. press 'Search for Book'

3. Using radio buttons (Below Clear button) to select one of 5 book returned.

4. Then 'Press here to Select Book' to enter into Saker database.

5. If none of the books match re-enter search term (step 1) and search again

(step 2)

6. Press the 'Clear' to return to beginning

View top 50 books entered in Saker Database

1. Press 'Clear' to return to start

2. Press 'View Library'

3. Top 50 books will be displayed (number of readers recommending "count")

4 Press 'Clear' to return to start

View a non top 50 book in Saker database

1. Press 'Clear' to return to start

2. Enter in title,author,isbn entry area

3. Press view Library

4. all matching books in Saker database will be returned

5 Press 'Clear' to return to start

Enter books in your Library/BookShelf

A library can be subdivided by entering a 'Bookshelf Name'

1. Press 'Clear' to return to start

2. Enter in title,author,isbn entry area

3. Enter your library name in 'My Library'

4. Press 'Search for Book

5. Select book using radio buttons

6. Then 'Press here to Select Book' to enter into database.

7. if a 2nd book is to be added

a. Enter in another book in title,author, isbn

b. repeat steps 4-6

8. Press 'Clear' to return to start

Delete books from your Library

1. Enter in your library name 'My Library'

2. Press the 'Delete Book' key

3. select the books to delete by clicking on checkbox on far right of book

4. Press 'Delete Book'

5. Press 'Clear' to return to start
